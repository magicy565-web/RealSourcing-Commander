# RealSourcing Commander — AI 能力全景评估

> **核心结论**：在单人创业的约束下，12 个 Agent 中有 **4 个** 可以做到"生产可用"，**4 个** 可以做到"演示可用"，另外 **4 个** 应该果断放弃。

---

## 1. 现有代码库的真实状态

经过对 `agentWorker.ts` 的深度盘点，我们必须诚实地面对一个现实：

**目前所有 12 个 Agent 的 AI 能力，都停留在"模拟模式"。**

它们的核心逻辑是调用 LLM 生成**看起来真实**的 JSON 数据，然后存入数据库。这对于前端开发和 UI 展示是完美的，但距离生产环境的"真枪实弹"还有很远的路。

| 真实状态 | 代码证据 |
| :--- | :--- |
| **无真实数据抓取** | 没有一行 `Playwright` 或 `Puppeteer` 代码 |
| **无真实 API 对接** | 没有一行 `HeyGen`, `TikTok API`, `Stripe` 的调用 |
| **大量模拟数据** | `agentWorker.ts` 中有 17 处 `模拟/mock/fallback` 标记 |

唯一的例外是 `autoReplyAgent`，它是我在 V3 冲刺计划中为你专门编写的，是**第一个真正意义上的生产级 Agent**。

## 2. AI 功能的三层分级：可落地 vs 不可落地

作为单人创业者，你的时间和精力是你最宝贵的资源。你不可能同时追逐 12 个目标。因此，我们必须对这 12 个 Agent 进行残酷的优先级排序。

### **第一层：生产可用 (Production-Ready)**

这些是你的核心资产，是用户愿意付费的根本。它们的技术实现相对简单，不依赖复杂的第三方 API 或爬虫，单人完全可以维护。

| Agent | 核心 AI 功能 | 单人可行性 |
| :--- | :--- | :--- |
| **08. 自动回复** | 读取询盘 → LLM 生成草稿 → Resend 发送 | **极高** (代码已完成) |
| **03. 选题助手** | 读取爆款数据 → LLM 生成脚本 | **高** (依赖 Kalodata 手动上传) |
| **09. 财务预算** | 读取收款记录 → LLM 分析财务健康度 | **高** (依赖 Stripe/PayPal Webhook) |
| **12. SEO 优化师**| 读取关键词排名 → LLM 生成优化建议 | **中** (依赖 Ahrefs/SEMrush 手动上传) |

### **第二层：演示可用 (Demo-Ready)**

这些功能非常吸引眼球，是绝佳的营销亮点。但它们的工程实现极其复杂，单人无法长期维护。正确的策略是：**用"Wizard of Oz"的方式实现它们，即手动操作 + AI 包装。**

| Agent | 核心 AI 功能 | 单人可行性 (生产级) | "Oz" 方案 |
| :--- | :--- | :--- | :--- |
| **02. 爆款雷达** | 自动抓取 TikTok 视频数据 | **极低** (反爬地狱) | **手动下载 Kalodata CSV** |
| **01. 线索猎手** | 自动抓取社媒评论 | **极低** (反爬地狱) | **手动下载 Apify 结果** |
| **04. 数字分身** | 脚本转数字人视频 | **中** (API 昂贵且不稳定) | **手动上传脚本到 HeyGen** |
| **05. 全网分发** | 自动发布视频到多平台 | **极低** (API 限制严格) | **手动发布，AI 生成文案** |

### **第三层：果断放弃 (Deprecate)**

这些功能要么价值模糊，要么技术实现严重依赖不可控的第三方，对于单人创业者来说是纯粹的"天坑"。

| Agent | 核心 AI 功能 | 放弃理由 |
| :--- | :--- | :--- |
| **06. 私信客服** | 自动回复私信 | **风险太高**，AI 幻觉可能导致商业纠纷 |
| **07. 邮件跟进** | 自动多轮邮件序列 | **复杂度太高**，状态机管理是噩梦 |
| **10. 物流管理** | 实时追踪全球物流 | **API 成本太高**，17Track 等服务按次收费 |
| **11. 阳光政务** | 自动检测合规风险 | **价值模糊**，目标客户群体不清晰 |

## 3. 最终建议：聚焦 4+4，形成组合拳

作为单人创业者，你的最佳策略是：

1.  **主打 4 个生产级 Agent**：把自动回复、选题助手、财务预算、SEO 优化师做到极致，这是你的收费基础。
2.  **包装 4 个演示级 Agent**：用"人工+智能"的方式，把爆款雷达、线索猎手、数字分身、全网分发作为你的营销亮点，吸引眼球，获取早期用户。
3.  **忘记剩下 4 个 Agent**：在你的产品 UI 和市场宣传中，彻底移除它们，避免分散精力。

这样，你最终能交付的是一个拥有 **8 个核心 AI 功能**的强大产品，既有扎实的生产能力，又有惊艳的演示效果，同时将工程复杂度控制在单人可承受的范围之内。
